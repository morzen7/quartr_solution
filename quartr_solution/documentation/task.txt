An important part of life as an engineer in the Data Automation team is developing new and
maintaining different API integrations towards our data providers.
A good example of this would be the famous SEC (U.S. Securities and Exchange
Commission) and their API. In this assignment you need to fetch the 10-K report, an annual
report that the companies need to submit to the SEC.
In order to accomplish this you have to write the implementation that will pull the latest 10-K
reports from some of the world's leading companies.
● Apple
● Meta
● Alphabet
● Amazon
● Netflix
● Goldman Sachs
You may find the documentation on how the SEC API works here and here. This logic would
be the basis for a service that fetches reports and exposes this data to other teams in Quartr.
So given a list of companies you need to fetch the latest 10-K reports from each company
and convert them to PDF format.

The submission must be written in Python and should contain the following:
● Source code with instructions on how to run the code
● The prompt log if any AI tool was used

https://www.sec.gov/search-filings/edgar-search-assistance/accessing-edgar-data
https://www.sec.gov/search-filings/edgar-application-programming-interfaces
